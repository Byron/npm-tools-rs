Utilities to help working with NPM repositories. [Click here](https://byron.github.io/npm-tools-rs/fs_utils/index.html) for the API-docs.

# Installation

In you `cargo.toml`:

```toml
[dependencies]
npm-tools = "*"
```

In your main module:

```rust
extern crate npm_tools;
```
